# cn-cns-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容各异，不针对任何特定域名或具体网站，仅作为静态资源的集中存放与展示。

## 项目简介

cn-cns-lab 是一个轻量级的静态页面仓库，主要存放独立的 HTML 文件及其相关资源。仓库本身不提供动态服务，所有页面均为前端静态实现，可通过 GitHub Pages 或其他静态托管服务直接访问。

## 目录说明

```
.
├── pages/          # 存放各个独立 HTML 页面及其子资源
├── assets/         # 公共资源文件（如样式、脚本、图片等）
├── index.html      # 仓库入口页面（如存在）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹代表一个独立页面项目，包含其 HTML、CSS、JavaScript 及所需素材。
- `assets/` 用于存放多个页面共享的公共资源，避免重复存储。
- 根目录下的 `index.html` 可作为导航或入口页，非必需。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 每个页面应保持自包含，引用资源路径建议使用相对路径。
- 页面可包含简单的交互功能（如 JavaScript 实现的表单、动画等），但不应涉及用户数据上传或存储。
- 归档时请确保页面在主流浏览器（Chrome、Firefox、Edge）中正常显示。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面，请将页面文件放入 `pages/` 下新建的目录中。
- 请确保新页面不包含恶意代码、跟踪脚本或违反平台政策的内容。
- 若需修改或删除已有页面，请说明理由。
- 仓库维护者保留对内容的审核与调整权限。
